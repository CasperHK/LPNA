# 🧭 全方位生涯導航系統 (Life-Path Navigator AI)
「數據驅動決策，消滅資訊差，讓每份努力都精準兌現。」
這是一個結合 AI 技術的終身成長輔助系統。從中學成績追蹤開始，延伸至大學選科、職場紀律分析及專業進修建議，旨在為使用者規劃最平穩、最高效的晉升路徑。

## 🎯 核心使命：少走彎路
* 預見風險：在成績下滑或職業停滯前發出預警。
* 消除盲目：根據實際數據（而非直覺）推薦最合適的學系與行業。
* 動態修正：隨時根據最新的文憑證照、工作表現，重新計算最佳路徑。

---

## 🚀 系統模組

### 第一階段：學術導航 (Academic Pilot)
* 成績軌跡分析：每年自動提取校內/DSE 成績，利用時序模型預測公開試表現。
* 精準選科推薦：比對全港院校收生大數據 (LQ/Median)，生成「穩妥、進取、衝刺」建議單，避免選錯科導致的分數浪費。
* 學科弱點掃描：AI 分析分數分佈，提示哪些科目是進入目標心儀學系的「攔路虎」，並提供補強建議。

### 第二階段：職涯進化 (Career Evolution)
* 職場紀律評估：整合入職後的表現數據（如出勤、考核、專案完成度），分析個人穩定性與執行力風格。
* 專業增值引擎：當使用者取得新的文憑 (Diploma) 或 專業證照 (Certification)，系統自動掃描市場需求，更新職業晉升勝率。
* 跨領域轉型導航：結合「原有專業 + 新修學位」，計算出最高溢價的轉行路徑（例如：會計背景 + Data Science 文憑 = Fintech 專家）。

---

## 🛠️ 技術架構
* 核心大腦：LLM (GPT-4 / Claude) 負責邏輯推理及職業規劃文案。
* 預測模型：Random Forest / XGBoost 用於預測分數趨勢與錄取機率。
* 數據安全性：採用最高級別的去識別化處理，確保個人成長數據不外洩。

## 📈 成長路徑示例
* 中四：系統發現數學極強但物理偏弱，建議選修經濟而非工程。
* 大三：分析 GPA 與實習表現，建議考取 FRM 證照以利進入金融業。
* 入職三年：記錄到用戶紀律極高且取得了 AI 碩士，建議從傳統後勤轉向 AI 產品經理。

## 📅 開發藍圖 (Roadmap)
* Phase 1: DSE 成績分析與 JUPAS 推薦 MVP。
* Phase 2: 大學 GPA 追蹤與實習匹配功能。
* Phase 3: 職場表現 (KPI) 與進修建議模組。
